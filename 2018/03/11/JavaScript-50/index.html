一、找出元素 item 在给定数组 arr 中的位置，如果数组中存在 item，则返回元素在数组中的位置，否则返回 -1

function indexOf(arr, item) {
    return arr.indexOf(item);
    }

indexOf() 方法返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1。

语法：arr.indexOf(searchElement)
      arr.indexOf(searchElement,fromIndex)

参数：searchElement 要查找的元素
      fromElement 开始查找的位置

返回值：首个被找到元素所在数组的索引位置，若是没有找到则返回-1

indexOf()详细用法

二、计算给定数组 arr 中所有元素的总和（数组中的元素均为 Number 类型）

var arr = [1,2,3,4];

方法一：常规循环
        function sum(arr) {
            var s = 0;
            一：
            for(var i=arr.length-1;i>=0;i--) {
                s += arr[i];
            }
            二:
            for(var i=0; i<arr.length; i++) {
                s += arr[i];
            }
            return s;
        }

方法二：函数式编程 map-reduce
        function sum(arr) {
            return arr.reduce(function(prev,curr,index,array) {
                return prev + curr;
            })
        }

    reduce()方法接收一个函数作为累加器（accumulator),数组中的每个值（从做到右）开始缩减，最终为一个值。   
    语法：arr.reduce((prev,curr,index,array) =>{},initialValue)
    参数：prev(previousValue) 初始值或是上一次调用回调函数返回的值
          curr(currentValue) 数组中当前被处理的元素
          index（currentIndex) 当前被处理元素在数组中的索引
          array 调用reduce的数组
          initialValue 可选参数，作为第一次调用callback的第一个参数
    返回值：最后一个调用回调函数返回的结果

reduce(详细用法)

方法三：forEach遍历
            function sum(arr) {
                var s = 0;
                arr.forEach((value,index,array) => {
                   s += value; 
                },0);
                return s;
            }

    forEach()方法对数组的每个元素执行一次提供的函数

        语法：
            array.forEach(callback(curr,index,array){
            // do something
            },this)
            array.forEach(callback[,thisArg])

        参数: callback 为数组中每个元素执行的函数，该函数接收三个参数
          curr(currentValue)当前值，数组正在处理的当前元素
          index索引，数组中正在处理的当前元素的索引
          array forEach()方法正在操作的数组
          thisArg 可选参数，当执行回调函数时用作this的值（参考对象）

forEach()详细用法

方法四：eval
            function sum(arr) {
                return eval(arr.join("+"));
            };

    eval()函数会将传入的字符串当做JavaScript代码进行执行
    语法：eval(string)
    参数：string 表示JavaScript表达式，语句或一系列语句的字符串。表达式可以包含变量以及已存在对象的属性
    返回值：执行指定代码之后的返回值，若返回值为空，则返回undefined

eval()详细用法

join() 方法将一个数组（或一个类数组对象）的所有元素连接成一个字符串并返回这个字符串
